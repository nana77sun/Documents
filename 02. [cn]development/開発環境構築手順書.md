### for windows

#### 開発ツールのインストール
 - gitをインストール  
 http://msysgit.github.io/

 - nodejsをインストール  
 http://nodejs.org/

#### ソース取得、および初期化
 - ソースコードを取得
>     # git clone https://github.com/r2space/YUKARiWeb

 - 依存ライブラリをインストール
>     # cd /workfolder/YUKARiWeb
>     # npm install

#### 起動、単体テスト、ソースコードチェック
 - アプリの起動
>     # node app.js

 - 単体テスト実行
>     # npm install -g jscoverage
>     # npm install -g mocha
>     # node tools/run_test.js

 - ソースコードチェック
>     # npm install -g plato
>     # plato -d reports -l .jshintrc modules/mod_company.js
